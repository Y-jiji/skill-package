This file tells you how to form a note about design choice. 

Step 1: Choice
+ Add `# Choice` section header. 
+ Write the design choice committed from [Reframed Question] as section body. 

Step 2: Alternatives
+ Add `# Alternatives` section header. 
+ Write other options from [Reframed Question] as section body. 

Step 3: Trade-off
+ Add `# Trade-off` section header. 
+ Write trade-off aspects as second-level subsections. 
+ If an aspect can be measured, give the metric which should be a real number in engineering. 

Step 4: Pin
+ Add `# Pin` section header.
+ State how user arrived at the design choice, and which sides the user commits to. 
+ How the `vars` and external references affect the aspects. When will the trade-offs be flipped?
