This file tells you how to form a note about yes/no statement. 

Step 1: Statement
+ Add `# Statement` section header. 
+ Write statement form of [Reframed Question] as section body. 

Step 2: Reason
+ Add `# Reason` section header. 
+ Add external references. 
+ How the `vars` and external references affect the answer. When will it be false?

Step 3: If the statement is contrary to prior believe/code, or it is about a bug:
+ Add `# Counter Example` section header.
+ If the statement is about something does not exist ...
    + Give an example, if there exists, then what contradiction it induce. 
+ If the statement is about something a counter example exists ...
    + Give one concrete construction. 
+ If the statement is about a bug ...
    + Paraphrase the bug logic
    + Give an example that triggers the bug
