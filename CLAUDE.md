## File Map

- `_AGENTS.md` — global CLAUDE.md source, installed to `~/.claude/CLAUDE.md`
- `AGENTS.md` — project-local agent instructions
- `claude.json` — settings + hooks config, installed to `~/.claude/settings.json`
- `Makefile` — install script for deploying to `~/.claude/`
- `hooks/` — hook scripts, installed to `~/.claude/hooks/`
- `skills/` — skill definitions, installed to `~/.claude/skills/`
- `plan/`, `note/` — working directories for propose/note workflows
- `tests/` — test suite
